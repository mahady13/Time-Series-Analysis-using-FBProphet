# 📈 Bitcoin Price Prediction & Time Series Analyzer

An interactive data science dashboard built with **Streamlit** and **FBProphet** to analyze, visualize, and forecast Bitcoin (BTC) prices. 

## 🚀 Features
* **Interactive Navigation:** Sidebar menu to jump seamlessly between dataset, preprocessing, charts, and forecasts.
* **Interactive Visualizations:** High-performance charting that supports hovering, panning, and zooming.
* **Machine Learning Predictions:** Implements FBProphet to forecast future price intervals and market direction.
* **Seasonality Breakdowns:** Identifies core macro trends and underlying weekly/yearly cyclical patterns.

## 🛠️ Tech Stack & Dependencies
* **Core Language:** Python 3.14+
* **Framework:** Streamlit
* **ML Model:** Prophet (FBProphet)
* **Data & Plots:** Pandas, Plotly, Matplotlib
